## 1. Getting Started

### Creating Your Account

**How to sign up:**
1. Visit https://tasknest.com/signup
2. Enter your email and create a password
3. Verify your email (check spam folder if not received within 5 minutes)
4. Complete your profile (name, company, role)
5. Invite team members (optional)

**Free Trial:**
- 14 days free trial (no credit card required)
- Full access to Professional tier features
- Can invite up to 10 team members
- No automatic billing after trial ends - you choose to upgrade

### First Steps After Signup

**Recommended setup (5 minutes):**
1. Create your first workspace (your team's hub)
2. Set up your first project
3. Invite team members via email
4. Create your first task
5. Explore templates (we have 50+ pre-built templates)

### Workspace Setup

**What is a workspace?**
A workspace is your team's central hub - think of it as your digital "nest" where all projects live. You can have multiple workspaces for different teams or departments.

**How to create a workspace:**
1. Click "+" icon in left sidebar
2. Select "New Workspace"
3. Name your workspace (e.g., "Marketing Team", "Product Development")
4. Choose privacy settings (Private/Public)
5. Invite members
6. Set workspace icon and color (optional)

**Workspace Limits:**
- Starter: 1 workspace
- Professional: 5 workspaces
- Enterprise: Unlimited workspaces

## 2. Core Features

### Projects

**Creating a Project:**
1. Go to your workspace
2. Click "New Project" button
3. Choose a template or start blank
4. Set project name and description
5. Assign project owner
6. Set deadline (optional)
7. Choose project color and icon

**Project Views:**
- **List View:** Traditional task list (default)
- **Board View:** Kanban-style boards with drag-and-drop
- **Calendar View:** Timeline and deadlines visualization
- **Gantt Chart:** Project timeline with dependencies (Professional tier+)
- **Table View:** Spreadsheet-like view with custom columns

**Project Templates:**
We offer 50+ templates including:
- Software Development Sprint
- Marketing Campaign
- Event Planning
- Product Launch
- Content Calendar
- Bug Tracking
- Sales Pipeline

### Tasks

**Creating a Task:**
1. Click "+" button or press "N" key (keyboard shortcut)
2. Enter task title
3. Add description (supports Markdown formatting)
4. Assign to team member(s)
5. Set due date and priority
6. Add tags and labels
7. Press Enter to save

**Task Properties:**
- **Status:** To Do, In Progress, Review, Done, Blocked
- **Priority:** Low, Medium, High, Urgent
- **Due Date:** Set deadlines with automatic reminders
- **Assignee:** Assign to one or multiple people
- **Subtasks:** Break down complex tasks into smaller steps
- **Attachments:** Upload files (up to 100MB per file)
- **Comments:** Team discussions on tasks
- **Time Tracking:** Log time spent (Professional tier+)
- **Custom Fields:** Add your own fields (Professional tier+)

**Task Dependencies:**
Link tasks so one task must be completed before another starts.
- Go to task details
- Click "Add Dependency"
- Select the task that must be completed first
- System will automatically adjust timelines

**Recurring Tasks:**
Create tasks that repeat automatically:
- Daily, Weekly, Monthly, Yearly
- Custom schedules (e.g., "Every 2 weeks on Monday")
- Set end date or number of occurrences

### Automation

**What is Automation?**
Automate repetitive workflows with rules and triggers. Save time on manual work.

**Common Automation Examples:**
1. **Auto-assign:** When task is created in "Bug" project → Assign to QA team
2. **Status updates:** When all subtasks are complete → Move parent task to "Review"
3. **Notifications:** When high-priority task is overdue → Notify project manager
4. **Recurring tasks:** Create weekly standup task every Monday at 9 AM
5. **Cross-project:** When task is marked "Done" in Project A → Create follow-up task in Project B

**How to create automation:**
1. Go to Project Settings
2. Click "Automations" tab
3. Click "Create Rule"
4. Set trigger (When... e.g., "Task is created")
5. Add conditions (If... e.g., "Priority is High")
6. Set action (Then... e.g., "Notify team")
7. Save and activate

**Automation Limits:**
- Starter: 5 automations per workspace
- Professional: 50 automations per workspace
- Enterprise: Unlimited automations

**Comments:**

- Reply to specific comments (threaded discussions)

**Real-time Collaboration:**

**Notifications:**
- In-app notifications (bell icon)

**File Sharing:**

- File size limit: 100MB per file
- Total storage: 10GB (Starter), 100GB (Professional), Unlimited (Enterprise)
- Version history for files (Professional tier+)

## 3. Account Management

### Profile Settings

**Update your profile:**
1. Click profile icon (top right corner)
2. Select "Settings"
3. Go to "Profile" tab
4. Update name, photo, timezone, language
5. Set working hours (for smart notifications)
6. Save changes

**Password Reset:**
If you forgot your password:
1. Go to login page: https://tasknest.com/login
2. Click "Forgot Password"
3. Enter your email address
4. Check email for reset link (expires in 1 hour)
5. Create new password (minimum 8 characters, must include number and special character)
6. If you don't receive email within 10 minutes, check spam folder or contact support

**Two-Factor Authentication (2FA):**
Secure your account with 2FA:
1. Settings → Security tab
2. Click "Enable 2FA"
3. Scan QR code with authenticator app (Google Authenticator, Authy, 1Password)
4. Enter 6-digit verification code
5. Save backup codes in safe place (you'll need these if you lose your phone)

### Team Management

**Inviting Team Members:**
1. Workspace Settings → Members tab
2. Click "Invite Members" button
3. Enter email addresses (comma-separated for multiple invites)
4. Select role (Admin, Member, Guest)
5. Add personal message (optional)
6. Send invitations

**User Roles:**
- **Admin:** Full access, can manage billing, members, and all settings
- **Member:** Can create projects and tasks, limited settings access
- **Guest:** View-only access to specific projects (free, doesn't count toward billing)

**Removing Team Members:**
1. Workspace Settings → Members
2. Find the member in list
3. Click "..." menu next to their name
4. Select "Remove from workspace"
5. Confirm action
6. Note: Their tasks will remain but become unassigned

**Transferring Ownership:**
To transfer workspace ownership to another admin:
1. Workspace Settings → Members
2. Find the new owner
3. Click "..." → "Transfer Ownership"
4. Confirm transfer (this action cannot be undone)

### Billing & Subscription

**Viewing Your Plan:**
Settings → Billing → Current Plan
You'll see: Plan name, billing cycle, next billing date, number of users

**Upgrading Your Plan:**
1. Settings → Billing
2. Click "Upgrade Plan" button
3. Select plan (Professional or Enterprise)
4. Enter payment details (credit card or PayPal)
5. Confirm upgrade
6. Prorated billing applies - you only pay for the remaining time in current cycle

**Downgrading Your Plan:**
1. Settings → Billing
2. Click "Change Plan"
3. Select lower tier
4. Confirm downgrade
5. Takes effect at end of current billing cycle (you keep current features until then)

**Payment Methods Accepted:**
- Credit/Debit cards (Visa, Mastercard, Amex, Discover)
- PayPal
- Bank transfer (Enterprise only, annual billing required)

**Billing Cycle Options:**
- Monthly billing
- Annual billing (save 20% - 2 months free)
- Billed per active user
- Guests are free (unlimited)

**Invoices & Receipts:**
Settings → Billing → Invoices
- Download PDF invoices for accounting
- View payment history
- Update billing information

**Important:** For refund requests, subscription cancellations, or billing disputes, please contact our billing team directly at billing@tasknest.com. AI support cannot process financial transactions.

### Setting Up Integrations

**General Steps:**
1. Settings → Integrations
2. Find the integration you want
3. Click "Connect" button
4. Authorize access (OAuth login)
5. Configure sync settings
6. Test the integration
7. Save settings

**Slack Integration (Step-by-Step):**
1. Settings → Integrations → Slack
2. Click "Add to Slack"
3. Select your Slack workspace
4. Choose channel for notifications
5. Configure notification types:
   - New tasks created
   - Task completions
   - Comments and mentions
   - Due date reminders
   - Project updates
6. Set up slash commands: `/tasknest create task` in Slack
7. Save settings

**GitHub Integration (Step-by-Step):**
1. Settings → Integrations → GitHub
2. Click "Connect GitHub"
3. Authorize TaskNest to access your repositories
4. Select repositories to connect
5. Link commits to tasks using: `#TASK-123` in commit message
6. Pull requests automatically create task comments
7. Commit history appears in task activity feed

**Google Drive Integration:**
1. Settings → Integrations → Google Drive
2. Click "Connect Google Drive"
3. Sign in with Google account
4. Grant permissions
5. Now you can attach Drive files directly to tasks
6. Files stay in Drive, only links are stored in TaskNest

## 5. Troubleshooting

### Common Issues & Solutions

**Issue: Can't log in / Forgot password**
**Solution:**
1. Click "Forgot Password" on login page
2. Enter your registered email address
3. Check email inbox (and spam folder)
4. Reset link expires in 1 hour
5. If no email received after 10 minutes, contact support@tasknest.com
6. Make sure you're using the correct email address

**Issue: Not receiving email notifications**
**Solution:**
1. Check Settings → Notifications → Verify "Email Notifications" is enabled
2. Check spam/junk folder
3. Add noreply@tasknest.com to your contacts/safe senders
4. Verify email address in Settings → Profile
5. Try "Send Test Email" button in notification settings
6. Check with your IT department if corporate email filters are blocking

**Issue: Tasks not syncing / Changes not appearing**
**Solution:**
1. Refresh browser (Ctrl+R on Windows, Cmd+R on Mac)
2. Check internet connection
3. Clear browser cache and cookies
4. Try different browser (Chrome, Firefox, Safari, Edge)
5. Check our status page: status.tasknest.com
6. Log out and log back in
7. If issue persists >10 minutes, contact support

**Issue: Can't upload files / Attachment failed**
**Solution:**
- Check file size: Maximum 100MB per file
- Supported formats: All common formats accepted
- Check internet connection stability
- Try compressing large files
- Check storage quota: Settings → Usage
- Try uploading smaller file first to test
- Clear browser cache
- Try different browser

**Issue: Integration not working (Slack, GitHub, etc.)**
**Solution:**
1. Go to Settings → Integrations
2. Disconnect the integration
3. Wait 30 seconds
4. Reconnect the integration
5. Check authorization hasn't expired
6. Verify permissions in the connected app
7. Check integration status page
8. Contact support with: integration name, error message, screenshot

**Issue: Mobile app not syncing**
**Solution:**
1. Pull down to refresh (swipe down gesture)
2. Check app is updated to latest version
3. Log out and log back in
4. Check phone's internet connection
5. Check phone storage (app needs space for cache)
6. Reinstall app if issue persists
7. Try on different network (WiFi vs mobile data)

**Issue: Forgot which email I used to sign up**
**Solution:**
- Try all your email addresses on login page
- Check your email inboxes for emails from noreply@tasknest.com
- Contact support with your name and company - we can help locate your account

**Issue: Account locked / Too many login attempts**
**Solution:**
- Wait 15 minutes before trying again
- Use "Forgot Password" to reset
- Contact support@tasknest.com if urgent

### Performance Issues

**App is slow / Laggy:**
**Solutions:**
1. Close unused browser tabs (TaskNest works best with <10 tabs open)
2. Clear browser cache: Settings → Privacy → Clear browsing data
3. Disable browser extensions temporarily (especially ad blockers)
4. Check internet speed (need minimum 5 Mbps for smooth experience)
5. Try incognito/private browsing mode
6. Update browser to latest version
7. Restart browser
8. Check CPU usage (close other heavy applications)

**Large projects loading slowly:**
**Solutions:**
- Archive completed tasks: Project Settings → Archive
- Use filters to view subset of tasks
- Split large projects into smaller projects
- Consider upgrading to Professional tier for better performance
- Use Board or Calendar view instead of List view for large projects

**Search is slow:**
- Wait for indexing to complete (happens after bulk imports)
- Use filters to narrow search scope
- Search within specific project instead of entire workspace

## 7. Security & Privacy

### Data Security

**Encryption:**
- **In Transit:** All data encrypted with TLS 1.3 (industry standard)
- **At Rest:** Data encrypted with AES-256 encryption
- **End-to-End Encryption:** Available for sensitive data (Enterprise tier)

**Data Centers:**
- Hosted on AWS infrastructure (SOC 2 Type II certified)
- Data residency options: US, EU, Asia-Pacific (Enterprise)
- 99.9% uptime SLA (Enterprise tier)
- Redundant backups across multiple availability zones

**Backups:**
- Automatic daily backups
- 30-day backup retention
- Point-in-time recovery available (Enterprise)
- You can export your data anytime

### Privacy

**Data Collection:**
- We collect only data necessary for service operation
- No selling of user data to third parties
- No advertising or tracking
- GDPR and CCPA compliant
- Full privacy policy: https://tasknest.com/privacy

**Data Export:**
Settings → Data & Privacy → Export Data
- Export all your data in JSON format
- Includes: tasks, projects, comments, attachments, activity logs
- Export ready within 24 hours
- Download link sent to your email

**Data Deletion:**
Settings → Data & Privacy → Delete Account
- Permanently deletes all your data
- 30-day grace period (can recover account within 30 days)
- After 30 days, data is permanently and irreversibly deleted
- Confirmation email sent

### Compliance & Certifications

**Certifications:**
- SOC 2 Type II certified
- ISO 27001 certified
- GDPR compliant (EU data protection)
- CCPA compliant (California privacy)
- HIPAA compliant (Enterprise with BAA - Business Associate Agreement)

**Access Controls:**
- Role-based access control (RBAC)
- Single Sign-On (SSO) - Enterprise tier
- SAML 2.0 support - Enterprise tier
- IP whitelisting - Enterprise tier
- Audit logs - Enterprise tier

**Security Best Practices:**
- Use strong passwords (min 12 characters recommended)
- Enable Two-Factor Authentication (2FA)
- Don't share your password
- Log out on shared computers
- Review connected integrations regularly
- Report suspicious activity immediately
